# 📘 Caderno Temático com IA - DIO

## 🧠 Contexto e Objetivo

Este projeto foi desenvolvido com o objetivo de utilizar a Inteligência Artificial como ferramenta de estudo.

O tema escolhido foi: **Fundamentos de Programação**.

Objetivos:
- Aprender conceitos básicos de programação
- Utilizar IA para melhorar o aprendizado
- Organizar o conhecimento de forma prática

## 🤖 Engenharia de Prompts

### Prompt 1:
"Explique o que são variáveis em programação"

Problema:
Resposta muito simples.


### Prompt 2:
"Explique variáveis em programação com exemplos simples e erros comuns"

Resultado:
Resposta mais clara e útil.



### Aprendizado:
Quanto mais detalhado o prompt, melhor a resposta.
